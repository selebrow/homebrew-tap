# Selebrow Tap

## How do I install these formulae?

`brew install selebrow/tap/<formula>`

Or `brew tap selebrow/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "selebrow/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
